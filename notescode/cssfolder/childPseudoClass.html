Pseudo-classes

pseudo-classes are keywords which allow selection based on information that lies outside of the document tree

    <style>

    .textcolor:nth-child(3){
        color:red;
    }

    </style>

    <div>

        <h1 class="textcolor">Awesome CSS And Text </h1>

        <p class="textcolor">Awesome CSS And Text </p>

        <p class="textcolor">Awesome CSS And Text </p>

        <p class="textcolor">Awesome CSS And Text </p>

    </div>

Output:

Awesome CSS And Text
