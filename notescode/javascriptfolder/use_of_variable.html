Variable

Use of var, let, const

var, let, const is variable. var and let use for sample variable used. const variable use for constant VARIABLES.

//use of var
var   firstname= "james" ;
console.log( firstname );

// use of let
let   secondname= "jones" ;
console.log( secondname );

// use of const
const   contry= "India" ;
console.log( contry );

Console output:

james

jones

India
