Spanning columns or rows

{% comment %} {/* window start */} {% endcomment %}

<!DOCTYPE    html>

<html lang="en">

{/* head start */}

    <body>

        <table>

            <tr>

                <td>row 1 col 1</td>

                <td>row 1 col 2</td>

                <td>row 1 col 3</td>

            </tr>

            <tr>

                <td colspan="3">These are convert second row and all are three columns</td>

            </tr>

            <tr>

                <td rowspan="2">This cell spans two rows</td>

                <td>row 3 col 2</td>

                <td>row 3 col 3</td>

            </tr>

            <tr>

                <td>row 4 col 2</td>

                <td>row 4 col 2</td>

            </tr>

        </table>

    </body>

{/* head end */}

</html>

{% comment %} {/* window end */} {% endcomment %}

Output:

row 1 col 1 row 1 col 2 row 1 col 3
These are convert second row and all are three columns
This cell spans two rows row 3 col 2 row 3 col 3
row 4 col 2 row 4 col 2
