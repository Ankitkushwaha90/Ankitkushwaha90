Lists

Ordered List

<ol> tag and each list can be created with the <li> tag:

For example

{% comment %} {/* window start */} {% endcomment %}

<!DOCTYPE    html>

<html lang="en">

{% comment %} {/* head start */} {% endcomment %}

    <body>

        <ol>

            <li>First Item</li>

            <li>Second Item</li>

            <li>onother Item</li>

        </ol>

    </body>

{/* head end */}

</html>

{% comment %} {/* window end */} {% endcomment %}

Output:

  1. First Item
  2. Second Item
  3. onother Item

Nested lists

You can list and then inner/sublist

{% comment %} {/* window start */} {% endcomment %}

<!DOCTYPE    html>

<html lang="en">

{% comment %} {/* head start */} {% endcomment %}

    <body>

        <ul start="4">

            <li>Item 1</li>

            <li>Item 2</li>

            <li> Item 3

            <ul>

                        <li>Item 3.1</li>

                        <li>Item 3.2</li>

                        <li>Item 3.3</li>

            </ul>

            </li>

        </ul>

    </body>

{% comment %} {/* head end */} {% endcomment %}

</html>

{% comment %} {/* window end */} {% endcomment %}

Output:

  • Item 1
  • Item 2
  • Item 3
    • Item 3.1
    • Item 3.2
    • Item 3.3
