Anchors and Hyperlinks

Parameter

Details

href

'href' is use for reder link of page. for example :
<a   href="Getstarted.com">Welcome  :)  </a>

rel

To denote that this link to an external website, you can use the external link type.
<a   href="Getstarted.com" rel="external">Hey    :)    </a>

target

Open link in new tab / window
<a   href="Getstarted.com" target="_blank">Hey    :)    </a>
