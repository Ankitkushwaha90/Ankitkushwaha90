file handling

how to open explorer windows.

Use of os.system in os module.

import os
//open explorer
os.system("explorer")

How to open c: directory or any directory.

Use of os.startfile in os module.

import os
//open c: directory
os.startfile("c:")
//open c: directory
os.startfile("c:/Program Files")
